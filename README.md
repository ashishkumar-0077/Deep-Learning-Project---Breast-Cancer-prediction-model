# 📌 Project Overview

This project uses an **Artificial Neural Network (ANN)** built with **TensorFlow/Keras** to classify breast cancer tumors as **Malignant (Cancerous)** or **Benign (Non-Cancerous)**.

The model is trained on the **Breast Cancer Wisconsin Dataset** and demonstrates how Deep Learning can be applied to healthcare for early disease prediction.

---

# 🎯 Objectives

- Predict whether a tumor is malignant or benign.
- Build a Deep Learning classification model.
- Understand neural network architecture.
- Evaluate model performance using various metrics.
- Apply preprocessing techniques for structured datasets.

---

# 📊 Dataset

- Dataset: Breast Cancer Wisconsin Dataset
- Source: Scikit-learn
- Total Samples: 569
- Features: 30
- Classes:
  - Malignant (0)
  - Benign (1)
